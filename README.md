# Binary Builder

Premium ta'lim platformasi demo loyihasi. Frontend, backend va lokal JSON database bitta Node.js server orqali ishlaydi.

## Ishga tushirish

```powershell
npm start
```

Keyin brauzerda oching:

```text
http://localhost:3000
```

## Demo loginlar

- O'quvchi: `student@binary.uz` / `binary123`
- O'qituvchi: `teacher@binary.uz` / `binary123`

## Asosiy imkoniyatlar

- Login, register, rol tanlash va profil rasmi
- O'quvchi va o'qituvchi dashboardlari
- Classroom yaratish, join code orqali qo'shilish
- Dars yuklash: text, PDF, rasm, video, YouTube, slayd, kodlash topshirig'i
- Test yaratish va avtomatik tekshirish
- Live dars UI: kamera, mikrofon, screen share, chat, raise hand, attendance
- 20 ta self-learning kurs
- Sertifikat generatori
- AI Study Assistant, XP, streak, badge, leaderboard, notification
- Dark/light mode va Uzbek/English/Russian til tanlash

## Tuzilma

```text
server.js          Backend API va static server
public/index.html  Frontend root
public/styles.css  Premium responsive UI
public/app.js      SPA interaktivligi
data/db.json       Lokal JSON database
```
